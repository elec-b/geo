## Description (≤4000)
Explorisで世界の国、首都、地理を学ぼう。インタラクティブな3D地球儀が宇宙から見た地球の姿を再現します。歪みなし、広告なし、トラッキングなし。

Explorisが違う理由

・本物の3D地球儀——国の本当の大きさを実感できます。平面地図では決して見せられないものです。
・6つのゲームタイプ——地図上で国を当てるところから、首都の場所を指すところまで。いろいろな方法で練習し、自分の限界に挑戦しよう。
・適応型アドベンチャーモード——あなたの理解度に合わせて出題し、弱点を強化し、さまざまなゲームを組み合わせて学習を助けるスマートなアルゴリズム。
・スタンプでいっぱいのパスポート——あなたの進捗はレベルと大陸のグリッドで形になります。レベルアップしながらパスポートを埋めていきましょう。
・国際連合が認める195か国——世界で最も広く受け入れられている基準。地球儀では係争中の地域や属領も探索できます。
・32言語——信頼できるソースから検証されたデータ：名前、首都、通貨、言語、住民名など。
・広告なし、個人データなし——あなたの進捗はあなたの端末にのみ残ります。オフラインで動作します。

使い方

探索——地球儀を回して、国をタップして、国旗、首都、人口、面積、通貨、言語、HDI、Wikipediaへの直接リンクが載った情報カードを見てみよう。

プレイ——アドベンチャーモードから始めよう。アルゴリズムがあなたの進捗に合わせた質問を出し、いろいろなゲームタイプを組み合わせて、退屈せずに学べるようにします。

パスポート——試験をノーミスでクリアしてスタンプを集めよう。各大陸でツーリストからバックパッカーへ、バックパッカーからガイドへと進もう。目標はパスポートをいっぱいにすること。

対象

8〜15歳の子どもやティーンエイジャー向けに設計されていますが、世界に興味のある大人にも夢中になれる内容です。

データソース

国際連合、Natural Earth、世界銀行、Wikidataから検証された公開データ。アプリはバックグラウンドで新しいデータを確認し、自動的に更新します。

プライバシー

Explorisは個人データを一切収集しません。アカウントなし、トラッキングなし、広告なし。あなたの進捗はあなたの端末にのみ残ります。

Explorisをダウンロードして、あなたのパスポートを埋め始めよう！
