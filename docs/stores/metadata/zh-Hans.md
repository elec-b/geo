## Description (≤4000)
使用 Exploris 学习世界上的国家、首都和地理：一个互动 3D 地球仪，还原从太空看地球的视角。没有变形失真，没有广告，也没有追踪。

为什么 Exploris 与众不同

· 真正的 3D 地球仪——感受国家的真实大小，这是平面地图永远做不到的。
· 6 种游戏类型——从在地图上辨认国家到指出首都的位置：用多种方式练习，挑战自己的极限。
· 自适应冒险模式——智能算法根据你的掌握程度调整出题，加强弱项，通过组合不同的游戏帮助你学习。
· 盖满印章的护照——你的进度化身为等级与大洲的网格。随着升级，逐步集满护照。
· 联合国承认的 195 个国家——全球最广泛接受的标准。你也可以在地球仪上探索争议地区和属地。
· 32 种语言——来自权威来源的已核实数据：名称、首都、货币、语言、居民名称等等。
· 无广告、无个人数据——你的进度只保存在你的设备上。可离线使用。

如何运作

探索——转动地球仪，点击国家，查看含国旗、首都、人口、面积、货币、语言、人类发展指数以及维基百科直达链接的信息卡片。

游戏——从冒险模式开始。算法会根据你的进度出题，并结合不同的游戏类型，让你在学习中不觉得无聊。

护照——通过零失误完成考试来获得印章。在每个大洲从游客升到背包客，再从背包客升到导游。你的目标：集满护照。

适合谁

为 8 到 15 岁的孩子和青少年设计，但内容引人入胜，连对世界充满好奇的大人也会沉浸其中。

数据来源

经过核实的公开数据来自联合国、Natural Earth、世界银行和 Wikidata。应用会在后台检查最新数据并自动更新。

隐私

Exploris 不收集任何个人数据。没有账户、没有追踪、没有广告。你的进度只存在于你的设备上。
