REVIEWER NOTES — Exploris

1. APP PURPOSE
Exploris is a geography learning app. Its core goal is to help users learn to locate the world's countries and their capitals on the map, using an interactive 3D globe instead of a flat map. The globe gives users a correct sense of each country's real surface area — something flat projections like Mercator distort heavily (Greenland and Africa looking similar in size is the classic example). The app combines free exploration of the globe with an adaptive quiz system that adjusts difficulty to the user's progress across six different question types.
Intended audience: kids and teenagers (8-15), and curious adults.

2. HOW TO ACCESS AND REVIEW THE APP
No account, no login, no credentials required. The app works fully offline after the first launch.
All core features are reachable from the bottom tab bar. Please set the app language to English in Settings before reviewing, so every label matches the names used below.
  · Explore — interactive 3D globe. Drag to rotate, pinch to zoom, tap a country to see its info card. Toggle to table view with the selector at the top.
  · Play — tap "Adventure" on any continent, or pick one of six specific game types. Correct/incorrect feedback is shown immediately after each answer.
  · Passport — grid of continents × levels showing earned stamps and locked cells.
  · Stats — per-country progress table.
  · Settings (gear icon) — language (32 languages), light/dark theme, haptics, markers toggle.
On first launch the app asks for a profile name and avatar; a profile is just a local label stored on-device, not an account.
For a more detailed overview of how the app works, tap the book icon in the top-left of the header to open the About screen.

3. EXTERNAL SERVICES / DATA SOURCES
Exploris uses only public, static geographic data from the sources listed below. There are no authentication, payment, analytics or AI services involved:
  · Natural Earth — country geometries (TopoJSON, 1:50m / 1:10m)
  · REST Countries v3.1 — country metadata
  · CLDR — country / currency / language names in 32 languages
  · Wikidata (SPARQL) — capital coordinates and Wikipedia slugs
  · UNDP Human Development Report — HDI values
  · World Bank Open Data — population and area
  · Wikipedia — linked externally from country cards (opens in Safari, not embedded)
All data requests use HTTPS. No user data is sent to any external service — no analytics, no tracking, no accounts.

4. REGIONAL DIFFERENCES
The app behaves identically in every region. On first launch it automatically picks the device language if it is among the 32 supported; otherwise it falls back to English. The user can change the language at any time from Settings. Features, content and data are the same in every language.

5. REGULATED INDUSTRY
Not applicable. Exploris is an educational app and does not operate in any regulated industry.

6. ACCOUNT / PAID CONTENT / UGC / SENSITIVE PERMISSIONS
None of these apply:
  · No account system — no registration, login or deletion flows.
  · No paid content, no in-app purchases, no subscriptions.
  · No user-generated content and no reporting/blocking features.
  · No prompts for location, contacts, camera, microphone or any other sensitive data or device capability.
